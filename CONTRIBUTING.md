# Contributing

Thanks for helping improve Grey's Express.

## Development

Use Java 21 and the included Gradle wrapper.

```powershell
.\gradlew.bat build
```

On macOS/Linux:

```sh
./gradlew build
```

## Pull requests

Keep changes focused, include tests when changing shared behavior, and update `README.md` or `CHANGELOG.md` when user-facing behavior changes.
